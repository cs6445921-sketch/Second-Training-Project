Log in# Colored Boxes 🎨

## Description
This is a simple Frontend project that displays multiple colored boxes on a web page.  
The project is designed to practice **HTML & CSS layout, styling, and responsiveness**.

---

## Features
- Multiple colored boxes displayed on the page
- Each box has a different background color
- Responsive layout: boxes adjust on different screen sizes
- Hover effect on boxes: changes opacity when hovered
- Easy to customize: add/remove boxes or change colors quickly

---

## Technologies Used
- HTML
- CSS

---

## Learning Goals
- Practice creating and styling elements with HTML & CSS
- Learn how to use CSS "Flexbox" or "Grid" for layout
- Understand responsive design principles
- Experiment with hover effects and colors
